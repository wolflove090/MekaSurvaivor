# カウガール攻撃間隔短縮 要件書

## 背景と目的
現在の `CowgirlStyleEffect` は空実装で、カウガールスタイルを選択しても固有能力が発生しません。  
本対応では、カウガール選択中のみプレイヤー攻撃の発動間隔を短縮し、他スタイルとの差別化と戦闘テンポの向上を実現します。

## スコープ内
- カウガールスタイル選択中、プレイヤーが持つ武器の攻撃間隔を一律で `75%` に短縮する。
- 他スタイルへ切り替えた時点で、攻撃間隔短縮を解除する。
- 既存のスタイル切替フロー（`PlayerController.ChangeStyle` -> `PlayerExperience.ChangeStyle`）に追従して効果を切り替える。
- 既に所持している武器と、カウガール選択中に新たに取得した武器の両方に短縮効果を適用する。

## スコープ外
- 武器ごとの個別バランス値の再調整。
- POW、弾速、範囲、バウンド回数など、攻撃間隔以外の性能変更。
- カウガール専用のUI演出、SE、VFX追加。
- 敵や召喚物など、プレイヤー武器以外の攻撃間隔変更。

## 機能要件
1. カウガール効果の適用
- `PlayerStyleType.Cowgirl` への変更時、プレイヤー攻撃間隔倍率を `0.75` に設定する。
- 効果適用後も `PlayerState.CurrentStyleType` はカウガールとして保持される。
- 同一スタイルを再適用しても倍率が重複乗算されない。

2. 攻撃間隔の計算
- プレイヤー武器の実効攻撃間隔は「武器ごとの基準攻撃間隔 × 現在の攻撃間隔倍率」で算出する。
- カウガール選択中は、基準攻撃間隔 `1.0` 秒の武器が `0.75` 秒で発動する。
- 倍率適用後の攻撃間隔は `0` 未満にならない。
- 既存の武器レベルアップで決まる基準攻撃間隔は維持し、その結果に対してカウガール倍率を乗算する。

3. 適用対象
- プレイヤーの通常武器システム (`WeaponBase` 派生武器) による攻撃に適用する。
- カウガール選択前から所持中の武器も、スタイル変更直後から短縮後の攻撃間隔で動作する。
- カウガール選択中に新規取得または再構築された武器も、生成直後から短縮後の攻撃間隔で動作する。
- `DroneWeapon` はドローン展開間隔のみ短縮対象とし、ドローン本体の `shotInterval` には補正を適用しない。
- 攻撃間隔の変更はプレイヤー武器に限定し、敵AIや敵弾のクールダウンには影響を与えない。

4. スタイル切替時の解除
- カウガールから他スタイルへ変更した時点で、攻撃間隔倍率を初期値 `1.0` に戻す。
- 効果解除後、プレイヤー武器の実効攻撃間隔は各武器の基準値へ戻る。
- スタイル切替を繰り返しても倍率の累積誤差や多重適用が発生しない。

## 非機能要件
- 既存コード規約（暗黙private、`_camelCase`、公開APIへのドキュメントコメント）を順守する。
- スタイル効果の責務は既存の `PlayerProgressionService` / `IPlayerStyleEffect` 系へ寄せ、UI層に攻撃間隔ロジックを持ち込まない。
- 攻撃間隔倍率は毎フレーム不要なアロケーションを発生させない方法で参照する。
- 攻撃間隔変更により既存クールダウン処理が破綻しないよう、スタイル切替直後の残りクールダウンも現在の有効間隔に収まるよう補正する。
- 参照未解決時は既存挙動を壊さず、倍率 `1.0` として扱える実装にする。

## 受け入れ条件
- カウガール選択中、基準攻撃間隔 `1.0` 秒の武器が `0.75` 秒間隔で発動する。
- カウガール選択前から所持していた武器にも、スタイル変更直後から攻撃間隔短縮が反映される。
- カウガール選択中に新しく取得した武器にも、取得直後から攻撃間隔短縮が反映される。
- カウガールから他スタイルへ変更すると、全てのプレイヤー武器の攻撃間隔が基準値に戻る。
- カウガールと他スタイルを複数回切り替えても、攻撃間隔倍率が二重に掛からない。
- 武器レベルアップによる基準攻撃間隔の変化と、カウガール効果 `0.75` 倍が両立する。

## 未確定事項・確認事項
- 攻撃間隔倍率は `PlayerState` に保持し、各武器の実効攻撃間隔計算時に参照する方針で確定。
